Practico 6 - Arrays

Ejercicio 1

Realizar una funcion que reciba un array que contiene strings. Retornar un nuevo array numerico cuyos elementos sean los largos de los strings del array original.





Ejercicio 2

Ingrese valores numericos en un array a partir de un campo de texto y un boton. Presionando otro boton mostrar en pantalla el mayor numero.





Ejercicio 3

Ingresar palabras e incorporarlas a un array indexado mediante un boton. Para finalizar presionar otro boton y mostrar un listado de todas las palabras almacenadas.





Ejercicio 4

Ingresar numeros e incorporarlas a un array indexado mediante un boton. Para finalizar presionar otro boton el promedio de los tados almacenados.





Ejercicio 5

Crear una funcion que reciba como parametro un array con valores numericos y devuelva un nuevo array solamente con los valores mayores a 20





Ejercicio 6

Una empresa necesita almacenar los alias (nombres de usuario) de red de sus empleados. Luego de cada ingreso mostrar todos los usarios ingresados hasta el momento.
Los alias no se deben repetir.





Ejercicio 7

a) Crear un campo de texto y un boton para ingresar nombres a un array. Cada ingreso mostrar todos los elementos, uno por linea, y al final mostrar la cantidad de elementos que tiene el array.
b)Crear un nuevo campo de texto y otro boton para ingresar en el campo de texto un numero de posicion, y mostrar en un parrafo el elemento que se encuentra es esa posicion. Ademas en otro parrafo mostrar el ultimo elemento del array.
c) Crear dos campos de texto para nombre a buscar y nuevo nombre y al presionar un boton buscar si ese elemento existe y en caso de existir sustituir el elemento buscado por el nuevo. Sino informar que no existe el elemento, en ese caso el nombre nuevo no se utilizara.











Ejercicio 8

Partiendo de un array [1,1] completar el array hasta la posicion 20 con los numeros de la sucesion fibonacci.



Ejercicio 9

Partiendo de un array que contenga los valores 1 y 1 en las primeras dos posiciones, completar el array con la secuencia de fibonacci hasta que el ultimo valor almacenado sea mayor a 1000, en ese momento detener el programa.
