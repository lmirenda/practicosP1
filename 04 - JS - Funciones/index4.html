Practico 5 - funciones

Ejercicio 1

Impelemantar una funcion que dados dos parametros formales numericos, n1 y n2, retorne cuantos numeros pares hay entre esos dos valores inclusive.







Ejercicio 2

Implementar una funcion esMayor(edad, tope) que devuleva "es mayor" si la edad es mayor al tope o "no es mayor" si no lo es







Ejercicio 3

Implementar una funcion "esBisiesto" que reciba como parametros un año y retorne true si es bisiesto, false en caso de lo contrario.





Ejercicio 4

Crear una funcion para calcular el area del triangulo que reciba como parametro la base y la altura.







Ejercicio 5

Crear una funcion para calcular el AreaRectangulo, que reciba como parametro ancho y alto y devuelva el area del rectangulo







Ejercicio 6

Crear una funcion que reciba como parametro un valor de temperatura y convierta ese valor. Celcius a Farenheit





Ejercicio 7

Crear una funcion que reciba como parametro un valor de temperatura en Farenheit y lo convierta a la unidad correspondiente





Ejercicio 8

Crear una funcion para calcular potencia, que reciba como parametro base y exponente y devuelva el resultado
